# Список тестов

## Класс Interpreter

- [x] : Тест суммирования введённых чисел : `let a: int = input(); let b: int = input(); print(a + b);`
- [x] : Тест отрицательной степени числа из ввода : `let a: int = input(); let b: int = input(); const result: int = -pow(a, b); print(result);`
- [x] : Тест вычисления площади круга по радиусу : `const radius: int = input(); const area: int = Pi * radius ** 2; print(area)`
