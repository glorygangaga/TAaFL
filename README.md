# Интепретатор языка программирования "CPY"

**CPY** - это императивный, статически типизированный язык программирования с C-подобным синтаксисом. Язык предназначен для обучения основам программирования и написания небольших программ.

###  Основные особенности
- **Статическая строгая типизация** - все типы проверяются на этапе компиляции
- **Императивная парадигма** - последовательное выполнение инструкций
- **С-подобный синтаксис** - знакомый и читаемый синтаксис
- **Блочная область видимости** - переменные видны только в своих блоках
- **Поддержка функций** - для организации сложных программ

**Что находится в репозитории?**
- /docs - спецификация языка программирования CPY, примеры кода на других языках программирования, их анализ, а также теория 
- /src - исходный код на C#, кроме тестов
- /tests - тесты к интерпретатору, модульные и приёмочные
- .editorconfig - конфигурационный файл для форматирования кода
- Directory.Build.props - общие параметры сборки
- compiler.sln - файл решения

**Как выполнить сборку и запустить тесты?**

Для того чтобы выполнить сборку необходимо:
1) Окрыть файл compiler.sln
2) Сделать <название решения> стартовым проектом
3) Запустить

Для того, чтобы запустить тесты необходимо:
1) Нажать CTRL+A или запустить через Test Explorer
