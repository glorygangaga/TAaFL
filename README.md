# Язык программирования "CPY"

##  Общее описание

**CPY** - это статически типизированный императивный язык программирования, разработанный для обучения и общего назначения. Язык сочетает в себе простоту синтаксиса с строгой типизацией, что делает его идеальным для изучения основ программирования.

###  Основные характеристики
- **Статическая строгая типизация** - все типы проверяются на этапе компиляции
- **Императивная парадигма** - последовательное выполнение инструкций
- **С-подобный синтаксис** - знакомый и читаемый синтаксис
- **Блочная область видимости** - переменные видны только в своих блоках
- **Поддержка структур и функций** - для организации сложных программ
